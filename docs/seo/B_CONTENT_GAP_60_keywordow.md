## Założenia metodologiczne (jak czytać tabele)

### Skąd dane volume?
Ahrefs Keywords Explorer (Polska, ostatnie 12 mc średnia). Walidowane Google Keyword Planner i Senuto.

### Skąd dane KD?
Ahrefs Keyword Difficulty (0-100 skala). Heurystyka: KD < 5 = łatwy quick win, 5-15 = średni horyzont, 15+ = wymaga budowania autorytetu.

### Co znaczy "konkurent rankujący"?
Domena z TOP 5 SERP w Polsce w czasie analizy (kwiecień 2026). Jeśli "brak" — żaden silny konkurent nie zajmuje top 3, łatwa fraza do przejęcia.

### Co znaczy "T1, T2, ..."?
Tygodnie zgodnie z roadmapą w `README.md`. T1 = pierwszy tydzień prac, T8 = ósmy tydzień, T9+ = rozbudowa po pierwszej fali.

## Wzorzec wewnętrznego linkowania

Dla każdego artykułu blogowego stosuj ten sam wzorzec linków wewnętrznych w treści:

1. **W pierwszym paragrafie:** 1 link kontekstowy do najbardziej powiązanej podstrony usługowej. Przykład w art. T3 "Adres URL": "Tworzenie stron internetowych w Poznaniu wymaga m.in. poprawnej struktury URL od pierwszego dnia."

2. **W środkowym paragrafie:** 1 link do innej podstrony usługowej lub artykułu blogowego. Przykład: "Jeśli interesuje Cię też pozycjonowanie SEO, polecamy nasz przewodnik..."

3. **W końcowym CTA:** 1-2 linki "Może Cię zainteresować" lub "Powiązane usługi" — do 2-3 najbardziej trafnych podstron.

4. **W stopce artykułu:** sekcja "Powiązane artykuły" — 3 linki do innych artykułów blogowych z tego samego klastra (np. cały klaster B).

**Cel:** każda nowa publikacja podnosi crawl rate i przepływ equity do podstron komercyjnych.

## Po każdej publikacji — checklist

- [ ] Tytuł = exact match keyword + benefit / number / year
- [ ] Meta description 150-160 znaków, zawiera keyword + CTA
- [ ] H1 zawiera keyword (bez stuffing)
- [ ] 5+ H2 logicznie strukturyzujące tekst
- [ ] Słowo kluczowe pojawia się w pierwszym paragrafie i naturalnie 5-10 razy w treści
- [ ] LSI keywords (synonimy, frazy pokrewne) w treści
- [ ] Min. 3 linki wewnętrzne (1 do podstrony usługowej, 2 do innych artykułów)
- [ ] Min. 1 link wychodzący do autorytetu (Wikipedia, dokumentacja, badania)
- [ ] Hero image WebP, alt-tag z keyword
- [ ] 2-3 grafiki w treści (Twoje, nie stockowe = unikatowe)
- [ ] Schema Article + BreadcrumbList
- [ ] Search Console "Sprawdź URL" + "Poproś o zindeksowanie"
- [ ] Repost LinkedIn + Facebook FOTZ Studio
- [ ] Po 14 dniach: sprawdź pozycję w Google incognito
